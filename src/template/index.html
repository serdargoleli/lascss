SCSS gücü

Map tabanlı ölçeklerle tamamen kontrol sizde. İstediğiniz utility'i ya da varyantı birkaç satırda ekleyin.

JIT motoru

Development sırasında sadece kullandığınız sınıflar üretilir, build süreniz ve çıktınız küçülür.

Uyumlu sözdizimi

Tailwind tarzı class isimleriyle öğrenme eğrisi neredeyse yok. Projeye hızlıca entegre edin.

Canlı deneme

Keyfi değerleri güvenli kullanın

Aşağıdaki alan, kullanıcı girdisini sınırlayan bir temizleyiciyle çalışır. Sadece harf, rakam, #, %, parantez, virgül, nokta, +, - ve boşluk içeren değerler kabul edilir.

Girdi bekleniyor…

Önizleme

Güvenli ve hızlı

Temizlenen değer direkt stil olarak uygulanır. İzin verilmeyen karakter gelirse, bu kutu varsayılan renkte kalır.

bg-blue-600
rounded-2xl
shadow-2xl
transition-all

Not: <script> gibi karakterler regex ile filtrelenir; kabul edilmediği için XSS tetiklemez.

Kullanım

NPM veya CDN

Kurulum

npm install las-css

import "las-css";

CDN

<link href="https://cdn.jsdelivr.net/npm/las-css@latest/dist/style.min.css" rel="stylesheet">

JIT için npm run jit çalıştırın, HTML/JS dosyalarınızı tarayıp public/las.css dosyasını otomatik oluşturur. Üretimde sadece kullandığınız sınıfları taşıyan küçük bir CSS elde edersiniz.
